# Currency Converter

## Description

The Currency Converter is a Java application that converts one currency into another using predefined exchange rates.

The project provides a menu-driven interface and supports multiple currency conversions.

## Features

* INR, USD, and EUR conversion
* Conversion history
* Input validation
* User-friendly interface
* Multiple conversions

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Arrays
* ArrayList
* Scanner Class
