# ATM Interface

## Description

The ATM Interface is a Java-based banking simulation project that allows users to perform basic banking operations through a console interface.

Users can check their account balance, deposit money, withdraw money, and validate transactions securely.

## Features

* Check Balance
* Deposit Money
* Withdraw Money
* Transaction Validation
* User-Friendly Menu Interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Classes and Objects
* Scanner Class
