# Student Management System

## Description

The Student Management System is a Java-based application designed to manage student records efficiently.

Users can add, view, search, update, delete, and sort student records through a menu-driven interface.

## Features

* Add Student
* View All Students
* Search Student
* Update Student Details
* Delete Student
* Sort Students
* Total Student Count
* Duplicate Roll Number Validation

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Collections Framework
* Scanner Class
