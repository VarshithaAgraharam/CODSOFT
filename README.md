# CODSOFT Java Development Internship

## Overview

This repository contains the projects completed as part of the CodSoft Java Development Internship. These projects demonstrate core Java programming concepts, Object-Oriented Programming (OOP), user interaction, data management, and problem-solving skills.

---

## Task 1 - Number Game

A Java-based guessing game where the user attempts to guess a randomly generated number within a limited number of attempts.

### Features

* Random Number Generation
* Limited Attempts
* Multiple Rounds
* Score Tracking
* User-Friendly Interface

---

## Task 2 - Student Grade Calculator

A Java application that calculates total marks, average percentage, and grade based on marks entered by the user.

### Features

* Input Marks for Multiple Subjects
* Calculate Total Marks
* Calculate Average Percentage
* Grade Assignment
* Result Display

---

## Task 3 - ATM Interface

A Java-based ATM simulation system that allows users to perform basic banking operations.

### Features

* Check Balance
* Deposit Money
* Withdraw Money
* Transaction Validation
* Menu-Driven Interface

---

## Task 4 - Currency Converter

A Java application that converts currencies using predefined exchange rates.

### Features

* INR, USD, and EUR Conversion
* Conversion History
* Input Validation
* Multiple Conversions
* Menu-Driven Interface

---

## Task 5 - Student Management System

A Java-based application designed to manage student records efficiently.

### Features

* Add Student
* View Student Records
* Search Student
* Update Student Details
* Delete Student
* Sort Students
* Total Student Count
* Duplicate Roll Number Validation

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP)

* Scanner Class
* Conditional Statements
* Loops
