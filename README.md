# CODSOFT Java Development Internship

This repository contains the projects completed as part of the CodSoft Java Development Internship.

## Projects

### Task 3 - ATM Interface
A Java-based ATM simulation system with deposit, withdrawal, and balance checking features.

### Task 4 - Currency Converter
A Java application for converting currencies such as INR, USD, and EUR.

### Task 5 - Student Management System
A Java application for managing student records with features like add, update, delete, search, and sort.
